# 🛡️ RoadShield India

**RoadShield India** is a modern frontend Progressive Web Application (PWA) that visualizes road accident hotspots across India through an interactive map and analytical dashboards.

Designed with a focus on user experience and data visualization, the platform allows users to explore state-wise accident-prone areas, view accident hotspots, and analyze accident statistics through interactive charts. The project demonstrates how modern frontend technologies can transform complex datasets into meaningful insights that promote road safety awareness.

> **Note:** This is a frontend-only demonstration project built using realistic mock datasets. The architecture is designed to support integration with official road accident datasets in the future.

### ✨ Key Features

* 🗺️ Interactive map of India with clickable states
* 📍 State-wise accident hotspot visualization
* 📊 Dynamic charts for accident trends and vehicle-wise analysis
* 🚗 Vehicle category classification (Car, Bike, Bus, Truck, Auto, etc.)
* ⚠️ Risk-based hotspot indicators
* 🔍 Search and filtering
* 🌙 Dark & Light Mode
* 📱 Fully responsive modern UI

**Built to demonstrate frontend engineering, interactive mapping, and data visualization while addressing the real-world challenge of road safety awareness.**
